# Dashboard Map Contracts

This document maps the human-readable dashboard layout to contract files.

## View Contract
- Default dashboard view: `contracts/dashboard/views.json`

## Widget Contracts
- System Health widget: `contracts/dashboard/widgets/system_health.json`
- Recent Decisions widget: `contracts/dashboard/widgets/recent_decisions.json`
- Drift Status widget: `contracts/dashboard/widgets/drift_status.json`

These are map contracts only and do not define UI implementation details.
